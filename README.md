# Idle RPG (Browser MVP Foundation)

A browser-based idle RPG progression game built with vanilla HTML, CSS, and JavaScript.

## Run

Open `index.html` in any modern browser.

## Constraints satisfied

- Browser-based
- No third-party software required
- No clicker-centric core loop (combat/progression is automatic)
- Uses only HTML/CSS/JS
- Save data persisted in IndexedDB with localStorage metadata
- Modular, system-driven architecture for easy expansion
